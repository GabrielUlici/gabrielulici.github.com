Jarvis

This is a work in progress. Some stuff are not working and I hope i will have time to fix them all.

Check the ChangeLog file to see what is working and check the Issue page to see what is not working.
The code was originally written by physicistjedi i modify it, added some stuff, made it to work and upload it to Github. The original page is here and the current page of development is here.

This my morning alarm clock.
Design question here is not how it looks like, or how it sounds like. The question is what do you want to hear in the morning in your bed. For obvious reasons it is called Jarvis.
Currently it gathers and reads the following:
  • Greetings.
  • Time and date.
  • Calendar events.
  • To-do list.
  • Weather conditions.
  • Unread email count and VIP email count.
  • Location of tracked packages.
  • News headlines.
  • Daily quotations.


Icon by Yellow Icon. Minimum Mac OS X version is 10.5 Leopard
